# Telegram Contact Bot 🤖

این ربات امکان ثبت تماس متنی، سفارش محصول و دریافت شماره تماس را فراهم می‌کند. اطلاعات از طریق ایمیل ارسال می‌شوند.

## ویژگی‌ها
- دکمه‌های تعامل با کاربر
- ذخیره اطلاعات ورودی
- ارسال ایمیل با SMTP
- استفاده امن از متغیرهای محیطی
